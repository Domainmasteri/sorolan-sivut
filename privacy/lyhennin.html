Privacy Policy for Sorola Linkkilyhennin

Effective Date: May 2026

1. Introduction

Sorola Linkkilyhennin ("the Extension") is committed to protecting user privacy. This Privacy Policy explains how the Extension handles information.

2. Data Collection and Usage

The Extension requires the activeTab permission to function. When the user clicks the Extension icon, it reads the URL of the currently active browser tab to send it to the user's configured srla.fi API endpoint for link shortening.

  • No Tracking: The Extension does not track your browsing history, monitor your behavior, or collect any analytics.
  • No Personal Data: No personal identification information, accounts, or cookies are collected or stored by the Extension.
  • No Data Retention: The URL of the active tab is processed instantaneously in real-time and is not cached or saved locally by the extension layout.

3. Data Sharing and Third Parties

We do not sell, trade, or rent user data to any third parties. Data is strictly transmitted to your own secure Cloudflare Worker endpoint (srla.fi) to create the short link.

4. Security

The Extension communicates exclusively via secure, encrypted HTTPS protocols. It does not load or execute any remote external scripts, ensuring that the code running on your browser is 100% local and safe.

5. Contact

If you have any questions regarding this privacy policy, you can contact us at: contact.make@sorola.fi
